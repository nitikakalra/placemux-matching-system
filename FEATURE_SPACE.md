# Student Features

- Java Score
- Python Score
- React Score
- Communication Score
- Projects Count

# Job Features

- Required Java Score
- Required Python Score
- Required React Score
- Required Communication Score
- Required Projects Count

# Matching Logic

A student is matched against a job using:

- Java Score
- Python Score
- React Score
- Communication Score
- Projects Count

If the student's score is greater than or equal to the job requirement, that feature is considered matched.

Match Score = Number of matched features / Total features × 100
